como publicar e manter atualizado
01
📊
Prepare o Google Sheets
Abra sua planilha → Arquivo → Publicar na web. Selecione a aba e formato Página Web.
Arquivo → Publicar na web → Formato: JSON
02
🔗
Cole a URL no Dashboard
Clique em ⚙ Planilha no topo. Cole a URL e configure o intervalo de refresh.
...spreadsheets/d/ID/gviz/tq?tqx=out:json
03
⚡
Auto-refresh ativo
O dashboard se atualiza automaticamente a cada 30 minutos. Sem necessidade de recarregar.
setInterval(refresh, 30 * 60 * 1000)
04
🌐
Publique online (grátis)
Netlify Drop — arraste o .html em app.netlify.com/drop.
GitHub Pages — suba como index.html.
app.netlify.com/drop → arraste → link ✓
💡 Dica pro: Para atualização em segundos (tempo real), use Google Apps Script para criar um webhook que dispara quando a planilha é editada.
